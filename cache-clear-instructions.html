🚨 キャッシュクリアが必要です！

⚠️ 重要

古いJavaScriptコードがブラウザにキャッシュされており、select=count*という不正なクエリが送信されています。

これを解決するには、ブラウザのキャッシュを完全にクリアする必要があります。

方法1: 🔥 スーパーリロード（推奨）

Windows/Linux:

  • Ctrl + Shift + R を押す
  • または Ctrl + F5 を押す

Mac:

  • Command + Shift + R を押す
  • または Shiftを押しながらリロードボタンをクリック

方法2: 🧹 開発者ツールからキャッシュクリア

  1. F12を押して開発者ツールを開く
  2. Networkタブを選択
  3. Disable cacheにチェックを入れる
  4. ページを再読み込み（F5）

方法3: 🗑️ ブラウザの設定から完全クリア

Chrome/Edge:

  1. Ctrl/Cmd + Shift + Deleteを押す
  2. 「キャッシュされた画像とファイル」を選択
  3. 期間を「全期間」に設定
  4. 「データを削除」をクリック

方法4: 🆕 プライベートウィンドウで開く

  • Chrome: Ctrl/Cmd + Shift + N
  • Firefox: Ctrl/Cmd + Shift + P
  • Edge: Ctrl/Cmd + Shift + N
  • Safari: Cmd + Shift + N

プライベートウィンドウでindex.htmlを開く

✅ キャッシュクリア後の確認

キャッシュクリア後、以下のボタンからアプリに戻ってください：

🚀 グルテンフリーマップに戻る

💡 ヒント

それでも問題が解決しない場合は、別のブラウザで試してみてください。
